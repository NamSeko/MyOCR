# MyOCR
Trong dự án này, tôi sử dụng mô hình OCR (Optical Character Recognition) để nhận diện và trích xuất văn bản từ hình ảnh. Mô hình này được huấn luyện trên một tập dữ liệu lớn các hình ảnh chứa văn bản.

Ở đây chúng tôi fine-tune mô hình `PaddleOCR` từ thư viện [PaddleOCR](https://github.com/PaddlePaddle/PaddleOCR). Dữ liệu huấn luyện trên một tập dữ liệu gồm các hình ảnh chứa văn bản tiếng Myanmar (Burmese).
